# To-do-list-React

## Description
To-do-list-React is a practical application that enables you to create lists of items to be done.
Organize your dayly or monthly schedule with us!

## Technologies used

1. HTML
2. CSS
3. JavaScript
4. React
5. Custom Hooks
6. Styled components
7. Redux-toolkit
8. Redux-Router
9. Redux-Saga
